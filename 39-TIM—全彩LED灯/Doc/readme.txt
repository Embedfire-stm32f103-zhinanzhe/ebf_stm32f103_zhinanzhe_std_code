【*】程序简介

-工程名称：全彩LED灯
-实验平台: 野火STM32-F103 指南者 开发板 
-ST固件库版本：3.5.0

【 ！】功能简介：
使用定时器输出控制LED灯展示RGB888的颜色


【*】注意事项：
	1.若移植本程序到其它硬件上时，请注意要选用定时器通道引脚来控制LED灯
	2.本实验与LED灯基础章节的主要差异是软件中的控制方式。
【 ！】实验操作：
编译并下载程序到开发板，复位后可以看到RGB灯亮各种颜色

/*********************************************************************************************/


【*】 引脚分配

RGB灯：
	R_LED	<--->PB5(TIM3 CH2)
	G_LED	<--->PB0(TIM3 CH3)
	B_LED	<--->PB1(TIM3 CH4)
		
											

/*********************************************************************************************/

【*】 程序描述
	
	<bsp_color_led.c>
	
		1.配置COLOR_TIMx复用输出PWM时用到的I/O
		2.配置COLOR_TIMx输出的PWM信号的模式，如周期、极性
		3.COLOR_TIMx 呼吸灯初始化
		4.停止pwm输出
		5.设置RGB LED的颜色 （一种设置颜色值格式、另一种设置颜色值）
